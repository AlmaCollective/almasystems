# Related Alma Repositories

This repository (`almasystems`) defines the **architecture and documentation** for Alma.

Experimental / implementation repos:

- `almasystems/alma-mvp` – early prototypes and demo experiments  
- `almasystems/alma-backend` – backend infrastructure experiments  
- `almasystems/alma-spiral-engine` – pattern & spiral engine experiments  
- `AlmaCollective/alma-core` – legacy core explorations  
- `AlmaCollective/alma-spiral` – legacy spiral explorations  
- `AlmaCollective/alma-emotion-ai` – emotional AI processing experiments
